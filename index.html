You have a cursor which you can move around using keyboard or via swiping.

You need to colour all the squares with the same colour.

You can change the colour of a square by making your cursor step on it.

The colour changes from F to E and so on to A after which it cycles back again to F.

If you feel that the puzzle is too easy then regenerate the puzzle as the generator algorithm is not perfect yet.
